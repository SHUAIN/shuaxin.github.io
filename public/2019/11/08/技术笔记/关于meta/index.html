关于meta

meta是什么

meta标签提供关于HTML文档的元数据。元数据不会显示在页面上，但是对于机器是可读的。它可用于浏览器（如何显示内容或重新加载页面），搜索引擎（关键词），或其他 web 服务。

作用

提供文档的描述信息

永远位于head内部

永远以键值对形式存在

组成

name

描述文档信息,以下列举常见参数

- 2.description---描述 - 3.viewport---移动窗口 - 4.robots---定义爬虫搜索方式 - 5.author---作者 - 6.generator---制作软件 - 7.copyright---版权 - 8.revisit-after---定义爬虫重访时间 - 9.renderer---双核浏览器渲染方式(360) ## http-equiv >http请求头作用，但在h5标准中不包括cache-control、Pragma、Expires，即使设置了也不会生效 - 1.content-Type---网页字符集 - 2.X-UA-Compatible---浏览器采取何种版本渲染当前页面 - 3.cache-control---请求遵循的缓存机制 - 4.expires---网页到期时间 - 5.refresh---自动刷新页面 - 6.Set-Cookie---设置cookie
